# ET-656 Platform

The **ET-656 Platform** is a mentorship and communication system designed to connect Compassion children with leaders. It provides real-time chat, secure communication, user role management, and a smooth, responsive interface.

This project includes a **React + Tailwind + Zustand** frontend and an **Express + MongoDB** backend.

---

## ⭐ Features
- One-on-one mentorship communication  
- Real-time chat system  
- Video call support (WebRTC or chosen tech)  
- User role management (Child, Leader, Admin)  
- User profile & session handling  
- Fast and responsive UI  

---

## 🛠 Tech Stack

### **Frontend**
- React  
- Tailwind CSS  
- Zustand (state management)

### **Backend**
- Node.js (Express)  
- MongoDB  
- JWT Authentication
